Вам дан список, сколько весят предметы для погрузки. Необходимо распределить их все так чтобы разница между общим весом в каждой руке была минимально возможной.

Входные данные: Список весов предметов, как список (list) целых чисел (int).

Выходные данные: Минимально возможная разница между весами в каждой руке, как целое число (int).

Примеры:

checkio([10,10]) == 0
checkio([10]) == 10
checkio([5, 8, 13, 27, 14]) == 3
checkio([5,5,6,5]) == 1
checkio([12, 30, 30, 32, 42, 49]) == 9
checkio([1, 1, 1, 3]) == 0

Связь с реальной жизнью: Это классическая комбинаторная оптимизационая задча по распределению. Комбинаторная оптимизация часто используется для составления сложных расписаний или просчета маршрутов.

Предусловия:
0 < len(weights) ≤ 10
all(0 < x < 100 for x in weights)
